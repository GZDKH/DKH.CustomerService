# DKH.CustomerService -- База данных

> Translation Pending — см. [английскую версию](../database.md)

## Обзор

DKH.CustomerService использует PostgreSQL в качестве основного хранилища данных через Entity Framework Core 10.0.2 с Npgsql 10.0.0. База данных следует схеме single-schema с фильтрами мягкого удаления на всех таблицах.
