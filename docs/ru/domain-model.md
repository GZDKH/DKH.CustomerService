# DKH.CustomerService -- Доменная модель

> Translation Pending — см. [английскую версию](../domain-model.md)

## Обзор

DKH.CustomerService управляет профилями клиентов, адресами доставки, списками желаний, предпочтениями и привязкой внешних идентификаторов для витрин Telegram Mini App. Домен следует принципам DDD с единым агрегатным корнем (`CustomerProfileEntity`) и тремя дочерними сущностями.
